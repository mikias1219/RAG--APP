Before you start

  1. Copy .env.example to .env and add Azure keys.
  2. Deploy models in Azure OpenAI: a chat deployment and an embedding deployment.
  3. Create the search index: python manage.py setup_search_index
  4. Index sample text: python manage.py index_documents
